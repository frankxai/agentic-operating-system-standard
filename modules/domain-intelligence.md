# Domain Intelligence Module

## Definition

Domain Intelligence helps operators turn Web2 and Web3 domain ideas into scored, verified, risk-gated buy, build, list, lease, hold, or pass decisions.

## Buyer

- Domain investor.
- Founder.
- Creator.
- Agency.
- Web3 identity operator.
- Portfolio holder.

## First-Win Workflow

```text
candidate list -> offline score -> verification queue -> agency swarm -> route -> report -> decision ledger
```

## Core Agents

| Agent | Output |
|---|---|
| Portfolio Strategist | Hold, sell, build, lease, or pass posture. |
| Name Miner | Candidate ideas and buyer thesis. |
| Comps Analyst | Comparable-sales evidence queue. |
| Brand Strategist | Naming territory and distinctiveness. |
| Web Designer | Build-first landing page brief. |
| Broker Lead | Venue and negotiation plan. |
| IP/Risk Sentinel | Trademark, UDRP, impersonation, and public claim review. |
| Web3 Specialist | ENS/Web3 owner, resolver, expiry, wrapper, and marketplace checks. |

## Required Skills

- Domain scoring.
- RDAP and registrar verification.
- ENS/Web3 verification.
- Brand naming.
- Comps research.
- Landing page strategy.
- Marketplace routing.
- IP and risk review.

## Approval Gates

- Purchase gate.
- Listing gate.
- Wallet gate.
- Trademark gate.
- Public claim gate.
- Customer data gate.
